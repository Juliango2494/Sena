LAS ETIQUETAS EN EL HTML SON LAS SIGUIENTES:

  • ESTRUCTURA BASICA:

  • <html>: Define el documento HTML.

  • <head>: Contiene meta-información sobre el documento (como enlaces a estilos o scripts).

  • <title>: Define el título del documento (se muestra en la pestaña del navegador).

  • <body>: Define el cuerpo del documento, donde se coloca el contenido visible.

  • ENCABEZADOS Y TEXTOS:

  • <h1> a <h6>: Define encabezados, donde <h1> es mas importante y <h6> el menos.

  • <p>: Define un parrafo.

  • <br>: Inserta un salto de linea.

  • ENLACES E IMAGENES:

  • <a>: Define un enlace.

  • <img>: Inserta una imagen.

  • <figure>: Contiene elementos multimedia como imágenes, con una posible leyenda.

  • LISTAS:

  • <ul>: Define una lista desordenada.

  • <ol>: Define una lista ordenada.

  • <li>: Define un elemento dentro de la lista.

  • TABLAS:

  • <table>: Define una tabla.

  • <tr>: Define una fila en una tabla.

  • <th>: Define una celda de encabezado en una tabla.

  • <td>: Define una celda de datos en una tabla.

  • FORMULARIOS:

  • <form>: Define una formulario.

  • <input>: Define un campo de entrada.

  • <button>: Define un botón.

  • <select>: Define una lista desplegable.

  • <option>: Define una opcion dentro de <select>.

  • ESTRUCTURA Y DIVISION DE CONTENIDOS:

  • <div>: Define una sección o contenedor.

  • <span>: Define un contenedor en línea.

  • <header>: Define el encabezado de una sección o página.

  • <footer>: Define el pie de página.

  • <main>: Define el contenido principal del documento.

  • SCRIPT Y ESTILOS:

  • <script>: Incluye o define un script (como JavaScript).

  • <style>: Define estilos CSS directamente en el HTML.

  • <link>: Vincula un archivo externo, como una hoja de estilos CSS.
